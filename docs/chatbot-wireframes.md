## 8. Accessibility Features

### Visual
- Minimum 16px font size for body text
- 18px+ for headings and key information
- WCAG AA contrast ratios (4.5:1 minimum)
- Focus indicators on all interactive elements

### Keyboard
- Tab navigation through all buttons and inputs
- Enter/Space to activate buttons
- Escape to close dialogs
- Arrow keys for quick-reply navigation

### Screen Readers
- ARIA live regions for new messages
- Role="log" on message container
- Button labels describe action, not just text
- Progress announcements ("Question 2 of 5")

### Motion
- Respects `prefers-reduced-motion`
- Typing indicator can be disabled
- No auto-scrolling that disrupts reading
